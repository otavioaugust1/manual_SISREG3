## CADASTRO
Ao acessar o menu Cadastro é possível navegar pelas opções: 

- OPERADORES
- ESCALAS
- AFASTAMENTO DE PROFISSIONAL
- AFASTAMENTO DE UNIDADE
- PREPAROS
- GESTÃO DE FILA DE ESPERA
- GESTÃO DE FILA DA REGULAÇÃO

## CONSULTA
Este menu é direcionado para realização de consultas, inclusive, algumas já descritas no perfil Administrador, conforme exposto abaixo. No entanto, diferente do perfil Administrador, as consultas realizadas pelo perfil Coordenador de Unidade trarão apenas os dados da unidade em que o usuário está vinculado.

O menu consultas do perfil Coordenador engloba consultas do menu Consulta Geral e Consulta AMB do perfil Administrador, com as seguintes funcionalidades: 

- CANCELAR SOLICITAÇÕES
- SOLICITAÇÕES
- AGENDA
- AGENDADOS PELA REGULAÇÃO
- DEVOLVIDOS PELA REGULAÇÃO
- AGENDADOS PELA FILA DE ESPERA
- PENDENTES PELA FILA DE ESPERA
- LIMITES/OCUPAÇÃO
- PPI/COTAS
- UNIDADE
- TETOS/UNIDADES
- PROFISSIONAIS
- GRUPOS/PROCEDIMENTOS
- TABELA SIGTAP
- PRONTUÁRIOS A ENVIAR
- PRONTUÁRIOS A RECEBER
- CNS
- ARQUIVO AGENDAMENTO (TXT)

## BPA
Ao acessar o menu BPA – Boletim de Produção Ambulatorial é possível navegar por duas funcionalidades: Geração de Arquivo (TXT) e Consulta BPA Gerado. Essas funcionalidades possuem uma limitação de horário para seu uso, estando bloqueadas das 8h às 15h. 

Essas funcionalidades serão visualizadas pelo coordenador de unidade solicitante, porém, ao tentar gerar ou consultar um BPA será apresentada na tela uma mensagem de erro informando que “este operador não possui permissão especial de acesso à manutenção de BPA”.
- CONSULTA BPA GERADO
- GERAÇÃO DE ARQUIVO
