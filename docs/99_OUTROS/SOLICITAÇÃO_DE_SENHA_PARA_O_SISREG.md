### **O Município para solicitar acesso ao sistema SISREG para regular procedimentos de média e/ou alta complexidade para seus munícipes e/ou ofertar esses procedimentos para outros Municípios com os quais possui pactuação (PPI),  deve  requerer a criação de um perfil  ADMINISTRADOR.**
Para que o sistema seja disponibilizado para o município é necessária manifestação do Gestor, formalizada por ofício para a Coordenação Geral de Regulação e Avaliação – CGRA,(escaneado e enviado por e-mail) com as seguintes informações: 

•	Descrever qual o tipo de Central de Regulação (ambulatorial e/ou hospitalar);

•	Indicação do ADMINISTRADOR com os seguintes dados: nome completo, CPF, e-mail, telefone, Unidade que deseja o cadastro (com o CNES). 

Vale ressaltar que os operadores com perfil ADMINISTRADOR devem realizar capacitação previamente à implantação, *preferencialmente realizada com técnicos da CGRA em Brasília.* Para verificar o cronograma do treinamento, acesse o Portal da Saúde.

### **O Município para requerer acesso ao sistema para solicitar procedimentos para outra Central de Regulação é necessária a criação de um perfil SOLICITANTE.**
Para o cadastro de operador solicitante é necessário ofício (escaneado) assinado pelo atual Secretário de Saúde para o email: sisreg@saude.gov.br. Informando no ofício os seguintes dados do operador: nome completo, CPF, email, telefone e qual a unidade que deseja o cadastro (com o CNES). O operador solicitante deve receber capacitação pela Central de Regulação para qual irá solicitar.

### **O operador solicitante esqueceu a senha – Centrais de Regulação que possuem apenas perfil SOLICITANTE.**
O operador envia um e-mail  para o e-mail sisreg@saude.gov.br,  informando: nome completo, login, CPF e a Central que está vinculado (IBGE ou código da central)

### **O operador esqueceu a senha – Centrais de Regulação que possuem perfil ADMINISTRADOR.**
O operador entra em contato com a própria Central a qual está vinculado para que o Administrador  faça o reset da senha.

### **O operador com perfil ADMINISTRADOR não trabalha mais na Central.**
Antes de se desvincular da Central de Regulação, o Administrador deve criar outro perfil Administrador e capacitar o próximo operador nas funcionalidades do sistema para que seja dada continuidade ao processo regulatório.

Caso o Administrador não trabalhe mais na Central de Regulação, é necessário o envio de oficio (escaneado), assinado pelo atual Secretário de Saúde para o email: sisreg@saude.gov.br, solicitando a desvinculação (inativação) do operador e o cadastramento, conforme orientação citada, acima.

### **Cancelar operador no SISREG**
Para o cancelamento (inativação) ser realizado é necessário e-mail informando: nome completo, login, CPF e a Central que esta vinculada (IBGE ou código da central)

### **Reativação de perfil no SISREG.**
Deve seguir a mesma orientação para o cadastro de operador.

 **Importante:** O Login e a senha para acesso ao SISREG é de uso pessoal e intransferível, caso o operador não trabalhe com o sistema é recomendável a inativação da senha.
