A **Lei Geral de Proteção de Dados (LGPD)** – Lei nº 13.709/2018 – estabelece regras sobre coleta, armazenamento, tratamento e compartilhamento de dados pessoais, com o objetivo de proteger os direitos fundamentais de liberdade e privacidade das pessoas naturais.

## Aplicação no SISREG

O SISREG trata dados de saúde classificados como **dados pessoais sensíveis** nos termos do art. 5º, inciso II, da LGPD. Todos os operadores devem observar:

- **Finalidade** — dados utilizados exclusivamente para fins de regulação em saúde.
- **Necessidade** — coletar apenas os dados estritamente necessários.
- **Segurança** — medidas técnicas para proteger os dados de acessos não autorizados.
- **Responsabilização** — gestores e operadores respondem pelo uso correto dos dados.

## Direitos do Titular

- Confirmação da existência de tratamento
- Acesso e correção dos dados
- Anonimização ou eliminação de dados desnecessários
- Portabilidade dos dados
