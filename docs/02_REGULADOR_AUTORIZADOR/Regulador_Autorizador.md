CONHECENDO O MENU

![MENU REGULADOR.png]({{ '/images/MENU%20REGULADOR.png' | relative_url }})

## AUTORIZAR
O menu Autorizar dispõe somente da funcionalidade Ambulatorial, quando o operador está habilitado apenas para essa função. Caso o operador esteja autorizado para o módulo internação o menu apresentará também a funcionalidade Internação.

- AMBULATORIAL
- INTERNAÇÃO

## SOLICITAR
O menu “Solicitar ” corresponde ao mesmo do perfil Solicitante. (Na funcionalidade Solicitar do Perfil Regulador os [parâmetros]({{ '/docs/01_ADMINISTRADOR/Administrador_Municipal/#CADASTRO_AMB' | relative_url }}) estabelecidos pelo [Perfil Administrador]({{ '/docs/01_ADMINISTRADOR/Administrador_Municipal/' | relative_url }}), não são fixos ou seja, caso o Regulador deseje solicitar um procedimento exclusivo do sexo feminino para um paciente do sexo masculino o sistema SISREG permitirá.)

- AMBULATORIAL
- INTERNAÇÃO

## CONSULTA GERAL
O menu “Consulta Geral” tem a função de propiciar ao administrador/operador consultas sobre um determinado Cartão SUS, quais as unidades cadastradas no [SISREG]({{ '/docs/99_OUTROS/SISREG_DEFINIÇÃO/' | relative_url }}), e consultar o Sistema de Gerenciamento da Tabela de Procedimentos, Medicamentos e OPM do SUS - SIGTAP.

- CNS
- UNIDADE
- TABELA SIGTAP

## CONSULTA AMB
Este menu é direcionado para realização de consultas. Ao acessar o menu “Consulta AMB” é possível navegar pelas seguintes opções: 

- SOLICITAÇÕES
- PPI/COTAS
- LIMITES/OCUPAÇÃO
- GRUPOS/PROCEDIMENTOS
- AGENDA
- TETOS/UNIDADES
- TETOS/PROCEDIMENTO
- COMPARATIVO PPI/TETO
- AGENDADOS PELA REGULAÇÃO
- DEVOLVIDOS PELA REGULAÇÃO
- AGENDADOS PELA FILA DE ESPERA
- PENDENTES PELA FILA DE ESPERA
- MÉDIA DE ESPERA EM FILA
- TRANSFERÊNCIA DE AGENDA
- CANCELAMENTOS
- SOLICITAÇÕES NÃO CONFIRMADAS / GERAL
- SOLICITAÇÕES NÃO CONFIRMADAS / UNIDADE
- SOLICITAÇÕES PENDENTES PELA FILA DE ESPERA
- BPA GERADO
- OFERTA DE VAGAS
- ARQUIVO AGENDAMENTO (TXT)

## CONSULTA HOSP
Este menu é direcionado para realização de consultas . No entanto, o administrador do sistema poderá realizar essas consultas em outras telas do menu “Cadastro hosp.” 

- MAPA DE LEITOS
- SOLICITAÇÕES
- PPI
- EXPORTADOR DE INTERNAÇÕES

## CADASTRO HOSP
Esse menu possibilita que o operador Regulador/Autorizador faça a configuração da central de regulação. Ao acessar o menu “Cadastro HOSP ” é possível navegar pelas seguintes opções: 

- IMPEDIMENTO DE LEITOS
