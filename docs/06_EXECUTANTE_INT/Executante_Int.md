Esse perfil permite ao operador executar as seguintes ações: 

·        Admitir a internação do paciente no sistema;  

·        Realizar as transferências de leitos; 

·        Registrar alta; e  

·        Solicitar mudanças de procedimentos no módulo hospitalar.
